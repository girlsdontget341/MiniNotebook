# MiniApp 记事本

这是一个简单的安卓记事本应用，可以帮助您快速记录想法、待办事项，并对它们进行分类管理。

## 主要功能

- **添加记事**: 快速添加新的记事内容，并可以为其指定一个分类。如果未指定分类，则会自动归入“未分类”。
- **删除记事**: 轻松删除不再需要的记事。
- **分类管理**: 为记事指定分类，方便整理和查找。
- **分类筛选**: 通过主界面顶部的下拉菜单，可以只查看特定分类下的记事。
- **数据持久化**: 所有记事都保存在本地的 SQLite 数据库中，即使关闭应用也不会丢失数据。

## 如何使用

1.  **添加**: 点击右下角的“+”按钮，在弹出的对话框中输入内容和分类，然后点击“保存”。
2.  **删除**: 在主界面，点击每条记事右侧的删除图标即可将其删除。
3.  **筛选**: 使用主界面顶部的下拉菜单来选择一个分类，列表将只显示该分类下的内容。
